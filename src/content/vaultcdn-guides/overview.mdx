VAULT is a security-focused CDN and media vault for small teams or personal use. It is not a generic file host — access is gated through Discord identity, upload permissions are controlled per user, and every file is stored with a UUID-based path that never trusts client-provided names.

The web app handles authentication, gallery management, in-browser image editing, and an admin panel. Files are stored on disk with metadata in MongoDB. Public URLs work without authentication so you can embed images, share links, and feed upload tools like ShareX.

An optional **Electron desktop app** wraps the same live site in a native Windows window — no separate backend, no feature gaps.

**Live instance:** [cdn.phxservices.co](https://cdn.phxservices.co)

**Desktop app:** [Download VAULT for Windows](https://github.com/Phoenix557/PhxCDN/releases/tag/desktop-v0.2.0)
