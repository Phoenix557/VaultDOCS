title: Scripts
description: npm scripts reference for PhxCDN.
order: 17
---

| Script | Description |
|--------|-------------|
| `npm run dev` | Start Next.js dev server |
| `npm run build` | Production build |
| `npm start` | Start production server |
| `npm run lint` | Run ESLint |
| `npm run migrate-uploads` | Migrate flat uploads to per-user folders |
| `npm run electron:dev` | Launch desktop app |
| `npm run electron:dist` | Build Windows installer |
| `npm run electron:smoke` | Automated Electron startup test |
