---
title: Admin Panel
description: Member management for admin users.
order: 11
---

Available at `/admin` for users in `ADMIN_DISCORD_IDS`.

- View all members with Discord avatars, storage used, and quota
- Search/filter the member table
- Toggle upload access per user
- Set storage quota (presets or custom)
- Add new members by Discord user ID before they have signed in
- Click a user to browse their uploads and delete files
- Env-configured allowlist/admins show a lock icon and cannot be deleted
